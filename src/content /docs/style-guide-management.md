# Managing Style Guide Categories

## Creating Categories

To create a new style guide category:

1. Click the **Change Type** dropdown in the Style Guide card.
2. Select **Create new category** at the bottom of the list.
3. Enter a **Label** for the category in the modal dialog.
   - The category key is automatically generated from the label (e.g., "My Category" becomes `MY_CATEGORY`).
4. Click **Create**.

## Category Management

The interface for editing and deleting categories has been removed. Category creation is now handled directly within the Change Type dropdown via a modal dialog.
