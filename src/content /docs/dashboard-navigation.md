# Analytics

View insights and metrics for your documentation synchronization workflow.

## Overview

The Analytics page provides visibility into your project's performance. It displays data for your active project, including metrics on suggestions, activity, and team efficiency.

## Prerequisites

To access the Analytics page, you must have:

1. A user account (GitHub OAuth)
2. An active organization
3. At least one active project configured

If no active project is found, the page displays setup instructions via the Setup Card component.

## Accessing Analytics

Navigate to the Analytics page from the dashboard sidebar:

1. Go to the Dashboard
2. Click **Analytics** in the navigation menu

The link is located in the main dashboard navigation, alongside Activity and other dashboard sections.
