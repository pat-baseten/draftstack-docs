## Creating Categories

The category creation interface has been simplified. When creating a new category, you only need to provide a **Label**.

### Fields

- **Label**: The display name for the category (e.g., "My Category").

### Automatic Key Generation

The **Key** is no longer a manual input field. It is automatically generated based on the Label you provide:
- The text is converted to uppercase.
- Spaces are replaced by underscores.

For example, if you enter the Label "My Category", the system automatically generates the Key "MY_CATEGORY".

### Description

The **Description** field has been removed from the creation and editing workflow.
