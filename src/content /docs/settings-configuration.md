# Managing Monitored Repositories

To add a repository to your project:

1. Click the **Add Repository** button.
2. A searchable picker will appear, displaying repositories accessible to your GitHub App installation.
3. Use the search bar to filter the list by repository name.
4. Click the **Plus** icon next to the repository you want to add.

The repository list displays the repository name in a monospace font. Private repositories are indicated by a "Private" badge.

If no repositories appear in the picker, ensure your GitHub App has the necessary permissions. You can update permissions via the "Update GitHub App permissions" link at the bottom of the picker.
