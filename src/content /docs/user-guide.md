# Making Manual Suggestions

You can manually trigger documentation updates by describing the change you need. DraftStack analyzes your description to identify the relevant documentation and generates the appropriate updates.

## Getting Started

To make a manual suggestion, you must have access to a project.

## Usage

1. Navigate to the **Dashboard**.
2. Click the **Make a Suggestion** button in the header.
3. Enter a description of the change in the modal.

### Description Requirements

The description field accepts between 10 and 5,000 characters.

```text
e.g., Rate limits changed from 50 to 100 requests per minute for all API endpoints...
```

4. Click **Submit**.

Once submitted, the suggestion is queued for processing. DraftStack determines which files to update and generates the changes.
