## Reworking Content

To refine a suggestion, use the "Steer with AI" section in the edit panel.

### Steer with AI

The "Steer with AI" section allows you to provide feedback to regenerate the content.

1. **Quick Presets**: Select from preset buttons such as "More concise", "Add code examples", or "Simplify language" to quickly populate the feedback field.
2. **Feedback**: Enter specific instructions in the text area (e.g., "Make the API section more concise, add rate limit details...").
3. **Active Style Guide**: If an active style guide exists, you can view it. Click "Active style guide" to expand the section. You can click "Edit for this rework" to modify the style guide specifically for this regeneration cycle. Click "Reset" to revert to the original style guide.
4. **Regenerate**: Click the **Regenerate** button to process your request. The button displays "Regenerating..." while the operation is in progress.

Upon success, a toast message confirms "Content regenerated".
